## 二、演示建议

1. 打开 `index.html`。
2. 点击「一键选两场」展示与参考截图接近的比分选择效果。
3. 点击「确认投注」。
4. 点击「随机」或指定主胜/平局/客胜开奖。
5. 切换到「气象站」展示魔法球、数据面板和历史轨迹。
6. 点击「自动模拟10轮」展示长期趋势。

如果要展示“多人实时版”，请进入项目目录运行：

```text
npm start
```

再打开：

```text
http://127.0.0.1:8765/
```

顶部显示“多人实时版”时，气象球读取的是服务器里的全站用户总盈亏。

## 三、课堂展示话术简版

这个作品把体育彩票的投注界面转化成一个“命运气象站”。观众在页面中进行虚拟投注，系统根据投注后的真实盈亏数据改变气象球颜色。暖色代表短期幸运，灰蓝和紫色代表返还率下降与集体亏损。作品想表达的是：赔率表并不是预测未来的工具，而是提前定价风险的系统。观众以为自己在选择比赛，实际上是在进入一个已经写好返奖率的数学结构。

## 四、注意事项

- 页面为艺术作品原型，不涉及真实金钱。
- 所有数据默认保存在当前浏览器本地；更换浏览器或清空缓存后会重置。
- 如果要重新开始演示，可以连续点击左上角彩色水晶球 3 次，然后确认重置。
- 如果直接双击页面后图片不显示，建议用本地预览地址打开。
- 如果要让微信好友或老师从手机访问，需要部署到公网 HTTPS 地址。
- 本地离线模式只能统计当前浏览器；多人实时模式才会统计所有用户。
